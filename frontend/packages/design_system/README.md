# design_system

Fondations visuelles partagées : **tokens, thèmes, composants et layouts**. Ne
contient **aucun métier** (issue #6).

## Contenu prévu

- tokens centralisés : couleurs, typographie, espacements, rayons, animations ;
- thèmes patient et hôpital ;
- composants accessibles et responsives ;
- catalogue de composants testable (widget tests, golden tests ciblés).

## Interdictions

- aucune dépendance métier ni appel réseau ;
- aucune couleur, taille, espacement ou animation codés en dur dans les
  applications : tout provient d'ici.
